# Neon AI Tic-Tac-Toe

A futuristic, arcade-style Tic-Tac-Toe game built with Python and Streamlit, powered dynamically by the Google AI Studio (Gemini Pro) Core. This application features a fully responsive, unbendable 3x3 grid layout optimized for mobile and desktop screens.

## Live Demo
Play the game live in your browser here:  
[Live Streamlit App](https://ai-tic-tac-toe-3la7xhuh8nfwwann3wv9cn.streamlit.app/)

---

## Features
* Google AI Studio Integration: Connects directly to the google-generativeai SDK to process strategic game moves.
* Dynamic AI Personalities: Switch between different AI behaviors (Sassy Trash-Talker, Friendly Coach, Evil Robot, Confused Genius) to change the AI's commentary style.
* Cyberpunk Neon UI: Custom CSS injection overrides standard browser styles to provide a glowing, dark-themed arcade grid layout.
* Secure API Key Handling: Built-in runtime sidebar input ensures private API keys are never hardcoded or leaked into the public GitHub repository.
* Offline Backup Mode: Automated random-move fallback mechanisms allow the game to remain fully playable even without an active API key.

---

## Project Architecture and Technologies
* Frontend/Deployment: Streamlit (Cloud Hosting)
* Backend Logic: Python 3
* AI Model Core: Google Generative AI SDK (gemini-1.5-flash)
* Styling Layer: Custom CSS Grid and Flexbox Injections

## Grading and Testing Instructions

To evaluate the live AI core capabilities:
1. Open the Live App Link: https://ai-tic-tac-toe-3la7xhuh8nfwwann3wv9cn.streamlit.app/
2. Expand the CONFIG sidebar menu on the left (or top on mobile devices).
3. Paste your official Google AI Studio API Key into the password field.
4. Select an AI Personality and begin placing your X marks on the board matrix!

Note: If no API key is provided, the system seamlessly triggers an instinctual offline backup routing protocol so the game loops never crash.
